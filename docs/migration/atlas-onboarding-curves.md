# Atlas onboarding curves

## Can we establish the curve?

**Draft documented structure: yes.**  
**Verified shortest-path curve: not yet** (1 verified platform; contract needs ≥3 comparable verified peers).

## What moves the needle

| Allowed | Forbidden |
|---|---|
| Verified audit counts (when eligible) | OpenRouter / taxonomy hypothesis links |
| Record `primary_path` / `friction_gates` counts (draft, labeled) | `heuristic_effort_score` |
| Median compare, per-signal below/at/above | Single difficulty / drop-off score |

## Scopes

1. **Corpus:** all eligible peers in the layer  
2. **Category:** same `category` (and for verified: same finish line + quality rules)

## API

`GET /api/platforms/:slug/curve` and journey payload field `curve`.
