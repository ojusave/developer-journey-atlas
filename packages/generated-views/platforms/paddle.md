2. Create a Paddle sandbox account as one opaque signup step. Success signal: The sandbox dashboard opens. Sources: S2, S3, S4.

5. Put the token and four price IDs into the official getting-started CodePen. Success signal: Starter and Pro prices render and the billing toggle works. Sources: S2.
6. Set the Default payment link under Checkout settings, using `https://localhost/` if no website exists. Success signal: The setting saves successfully. Sources: S2, S3.
7. Open checkout from CodePen and complete a sandbox payment with Paddle’s published test values. Success signal: Checkout completes successfully. Sources: S2, S3.
8. Open Hookdeck Console, copy its endpoint, then create a Paddle platform webhook destination for all events. Success signal: The destination saves successfully. Sources: S2.

10. Inspect the received Hookdeck events. Success signal: Hookdeck shows lifecycle events including `transaction.completed`. Sources: S2.
