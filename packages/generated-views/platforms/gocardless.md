## Documented path

1. Open the official Set up page and choose the Sandbox environment used by the guide. Success signal: The Set up step is visible. Sources: S1.
2. Sign up for or sign in to the GoCardless Sandbox. Success signal: The Developers tab is accessible. Sources: S1, S2.
3. Create a Read-Write sandbox access token. Success signal: The one-time token value is copied. Sources: S1.
4. Create a BACS Billing Request with cURL. Success signal: The response contains a Billing Request ID such as `BRQ123`. Sources: S1, S3.
5. Create a Billing Request Flow for the Billing Request. Success signal: The response contains a hosted authorisation URL. Sources: S3.
6. Open the hosted authorisation URL and complete the sandbox payer flow. Success signal: The hosted flow confirms completion. Sources: S3.
7. Open Customers in the sandbox dashboard and verify the new customer. Success signal: The newly created customer is visible on the Customers page. Sources: S3.
