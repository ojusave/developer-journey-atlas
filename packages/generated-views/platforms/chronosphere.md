## Documented path

1. Open Get started with Observability Platform and select the first developer task, Query telemetry data. Success signal: Query Builder is recommended for constructing PromQL. Sources: S1.
2. Obtain organization-provisioned access and authenticate at the tenant URL. Success signal: The authenticated home page is displayed. Sources: S1.
3. Open Metrics Explorer and click Builder for a query. Success signal: Metric, label, function, and rollup controls are available. Sources: S4, S5.
4. Select an available metric and any desired labels or rollup. Success signal: The expression appears in the query field. Sources: S4.
5. Click Run. Success signal: The graph updates and Results shows matching series and values in Graph and Table. Sources: S4, S5.
