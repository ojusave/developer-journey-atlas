## Documented path

1. Open the official Astro CLI installation guide. Success signal: The Homebrew installation route is documented. Sources: S4.
2. Run `brew install astro`. Success signal: The command completes successfully. Sources: S4.
3. Run `astro version`. Success signal: A valid Astro CLI version appears. Sources: S4.
4. Create an empty directory, enter it, and run `astro dev init`. Success signal: The standard project files and example DAG exist. Sources: S5.
5. Run `astro dev start --standalone` from the project directory. Success signal: Startup completes and prints local access information. Sources: S6.
6. Open `https://localhost:8080/` and inspect the generated example DAG. Success signal: The example DAG appears in the DAG list. Sources: S5, S6.
