**Developer goal:** Create a first OptiView Real-time account, then use the official dashboard-only tutorial to create a publish token, broadcast a webcam stream, and view the server round-trip in the hosted Viewer App.

**Meaningful success:** The developer broadcasts webcam audio and video from the OptiView dashboard and views the stream after its server round trip in the hosted Viewer.

**Evidence boundary:** explicitly-named. The tutorial ends Part 1 at hosted playback and then directs the developer to later custom web-app work.

## Documented path

1. Open Getting started with OptiView Real-time Streaming and select Part 1, Use the Streaming Dashboard. Success signal: Part 1 is labeled as the starting point for broadcasting and playback without code. Sources: S1, S2.
2. Sign up for or sign in to an OptiView Real-time account. Success signal: The Streaming tab is available. Sources: S1, S2, S3, S5.
3. Create a publish token. Success signal: The token appears in the Streaming dashboard. Sources: S2.
4. Start a dashboard broadcast. Success signal: The LIVE box and running timecode appear. Sources: S2.
5. Launch and view the hosted stream. Success signal: The hosted Viewer plays the full server round-trip stream. Sources: S2, S4.
