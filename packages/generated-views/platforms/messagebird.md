## Documented path

1. Sign up for a free MessageBird account and verify email. Success signal: Account active and accessible via Dashboard. Sources: S3.
2. Create an API access key in the MessageBird Dashboard. Success signal: API key visible in Dashboard. Sources: S3.
3. Purchase a virtual mobile number or verify an existing phone number. Success signal: Number listed in Dashboard. Sources: S3.
4. Send a message using the MessageBird API with the API key and verified number. Success signal: Message appears in Dashboard logs and API returns success. Sources: S3, S1.

## Documented friction gates

- Must create and verify a MessageBird account via email confirmation. Stage 1. Evidence state: documented requirement. Sources: S3.
- Must create an API access key in the Dashboard to authenticate API requests. Stage 2. Evidence state: documented requirement. Sources: S3.
- Must have test credits or top-up balance to send messages beyond own verified number. Stage 3. Evidence state: documented requirement. Sources: S3.
- Must have a verified phone number or purchase a virtual mobile number to send messages. Stage 3. Evidence state: documented requirement. Sources: S3.
- For WhatsApp Business onboarding, Facebook Business verification is required. Stage 1. Evidence state: documented requirement. Sources: S6.
- WhatsApp Business application approval by WhatsApp is required before channel activation. Stage 2. Evidence state: documented requirement. Sources: S6.
- WhatsApp channel activation requires entering a verification code sent to the phone number. Stage 4. Evidence state: documented requirement. Sources: S6.
