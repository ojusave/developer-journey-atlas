2. Copy the Space ID, create or copy an API key from the credential location shown by the selected current guide, and store both securely. Success signal: Both credentials are available for local configuration. Sources: S1, S2, S3.
3. Install `arize-otel`, the LLM provider SDK, and the matching OpenInference instrumentor. Success signal: The packages import in the application environment. Sources: S1, S3.
4. Register the tracer provider before any LLM call and invoke the instrumentor. Success signal: Instrumentation executes before the first model call. Sources: S1, S3.
5. Run representative requests through the instrumented application. Success signal: Requests complete and spans are sent to AX. Sources: S1.
6. Open the AX project, confirm the request traces, and expand one trace. Success signal: A trace corresponding to the run is visible with its execution details. Sources: S1.
