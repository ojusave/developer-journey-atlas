## Documented path

1. Open the AssemblyAI pre-recorded audio Quickstart. Success signal: The guide shows Transcribe your first file. Sources: S1.
2. Create an AssemblyAI account at the official signup URL. Success signal: The dashboard opens. Sources: S1, S2.
3. Open Workspace, Manage, API Keys; select Create New API Key, name it, create it, and copy it. Success signal: The key can be copied. Sources: S1, S3.
4. Export the key as `ASSEMBLYAI_API_KEY`. Success signal: The script can read the key. Sources: S1.
5. Run `pip install assemblyai`. Success signal: The install command completes. Sources: S1.
6. Save the published Python sample as `transcribe.py`. Success signal: The file contains the one-call transcription. Sources: S1.
7. Run `python transcribe.py` and wait for the SDK to upload, submit, and poll. Success signal: Output begins `Smoke from hundreds of wildfires...`. Sources: S1.
