## Documented friction gates

- Xcode-based development requires compatible Apple hardware and the selected simulator or device support. Stage 2. Evidence state: documented requirement. Sources: S2, S5.

- Project name, organization identifier, interface, language, testing, storage, platform, and run destination vary by template. Stage 2. Evidence state: documented requirement. Sources: S2, S3, S4.

- Paid program enrollment is not required for learning or simulator use, but distribution membership is listed at 99 USD per year where applicable. Stage 2. Evidence state: documented requirement. Sources: S5.
