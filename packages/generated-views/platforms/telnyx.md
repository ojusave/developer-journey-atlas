**Developer goal:** Create a Telnyx account, enable messaging on two numbers, and send a first SMS through the Messaging API.

**Meaningful success:** A POST to /v2/messages with a Bearer API key queues an SMS from one Telnyx number to another and returns a successful message payload.

**Evidence boundary:** explicitly-named. This is the terminal action of the official five-minute Send Your First Message guide.

## Documented path

1. Open Send Your First Message in the Telnyx developer docs. Success signal: Prerequisites and numbered steps are shown. Sources: S1.
2. Create a Telnyx account through the official signup entry linked from the guide, then sign in to Mission Control. Success signal: The portal is reachable after signup. Sources: S1, S2.
3. Purchase two Telnyx phone numbers for on-net messaging tests. Success signal: Both numbers appear under My Numbers. Sources: S1.
4. Open Messaging, create a messaging profile, and assign both numbers to that profile. Success signal: Each number's Messaging Profile dropdown shows the selected profile. Sources: S1.
5. Open API Keys and copy or create an API key. Success signal: The key can be pasted into the Authorization header. Sources: S1.
6. POST to https://api.telnyx.com/v2/messages with Bearer auth, from and to set to the two Telnyx numbers in E.164, and text Hello, world!. Success signal: The API returns a successful message response payload. Sources: S1.

## Documented friction gates
