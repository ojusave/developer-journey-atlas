## Documented path

1. Open the official Imgix Quick Start Guide. Success signal: Create an Imgix Account and Create a Source are shown. Sources: S1.
2. Sign up for Imgix as one opaque account step. Success signal: The Imgix dashboard opens. Sources: S1, S2.
3. Create a Source and select where the assets are stored. Success signal: Step 2 displays fields for the selected Source type. Sources: S1, S3.
4. Enter the storage details required by the selected Source type. Success signal: The subdomain step becomes available. Sources: S1, S3.
5. Choose a unique imgix.net subdomain. Success signal: The deployment review is available. Sources: S1.
6. Select Deploy Source and wait for successful deployment. Success signal: The dashboard reports successful deployment. Sources: S1.
7. Open the documented Imgix URL for a known source asset. Success signal: The expected source asset appears in the browser. Sources: S1.

## Documented friction gates

- Deploy Source queues the Source for deployment; the subdomain becomes active only after successful deployment. Stage 2. Evidence state: documented requirement. Sources: S1.
