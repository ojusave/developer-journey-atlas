## Documented path

1. Go to tailscale.com, select Get Started, and sign in with a supported SSO identity provider. Success signal: The Welcome to Tailscale page appears. Sources: S1.
2. Select Business use or Personal use. Success signal: Operating-system options are shown. Sources: S1.
3. Select the first device's OS, download and install the client, then authenticate with the same identity. Success signal: The device appears in the browser onboarding window. Sources: S1, S2.
4. Select the second device's OS, copy the provided link to that device, install the client, and authenticate it. Success signal: Both devices display in the onboarding page. Sources: S1.
5. Select Take me home and review the admin console. Success signal: The Machines view can show enrolled devices with their Tailscale identities and addresses. Sources: S1, S2.
