- AWS will not allow standalone signup to continue until a valid payment method is added. Stage 5. Evidence state: documented requirement. Sources: S4.
- AWS can require phone ownership verification by SMS during signup. Stage 6. Evidence state: documented requirement. Sources: S4.
- Account activation usually takes a few minutes and can take up to 24 hours. Stage 8. Evidence state: documented requirement. Sources: S4.
- AWS requires a choice among multiple services, workloads, and learning outcomes before one resource-oriented first mile exists. Stage 9. Evidence state: documented requirement. Sources: S1, S2.
